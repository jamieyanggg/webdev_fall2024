Relative
Absolute positioned divs look for a parent element with a defined position property to position against. If it can't find one, it positions absolutely to the body tag
Fixed behaves similary to absolute but will be fixed regardless of scrolling

When we have a relative parent, absolute or fixed position child element will position in relation to the relatively positioned parent.

absolute position relative parent
fixed position relative parent
